Use estas URLs para enviar eventos ao Tracker a partir de outras ferramentas (n8n, Make, Zapier, etc.). Faça uma requisição POST com o corpo em JSON.

  "utm_campaign": "campanha-1"
}
Parâmetros UTM — Facebook Ads

Adicione estes parâmetros na URL de destino dos seus anúncios para rastrear a origem dos cadastros e depósitos.

Parâmetros dinâmicos (Facebook preenche automaticamente)
utm_source=facebook&utm_medium={{placement}}&utm_campaign={{campaign.name}}&utm_content={{ad.name}}&utm_term={{adset.name}}
URL completa de exemplo
https://sua-plataforma.com/register?utm_source=facebook&utm_medium={{placement}}&utm_campaign={{campaign.name}}&utm_content={{ad.name}}
Como usar
1. Copie os parâmetros dinâmicos acima
2. No Gerenciador de Anúncios → edite o anúncio → URL do site
3. Cole após a URL da sua plataforma: https://plataforma.com/register?parâmetros aqui
4. O Facebook substituirá automaticamente os valores {"{{"}campo{"}}"} ao exibir o anúncio
